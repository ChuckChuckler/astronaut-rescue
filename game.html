Astronaut Rescue

Astronaut Rescue is a game where you play as an astronaut trying to save your ship.



This is the computer.

Here, you can access the the Quadra Database containing the equations you need to solve.

If you have minigames enabled, you'll need a password to access the database!

You can guess this password through a Wordle minigame.

If you can guess the password in five tries, yippee! You can now log in at anytime!

If not, the password will be changed, and you'll have to try again.

This is the safety button.

This is the feature that's pretty much your life or death.

As you've probably already gathered, there's a six-digit code required to unlock the button.

There's a key next to the input panel that provides the solutions of six of the equations in the Quadra Database.

If you find a letter that matches, type it in the input box.

If you have minigames enabled, you'll play a small minigame where you have to line up the letter with a space below it.


Enemy

If you have minigames enabled, there's a chance for an enemy to spawn in the sky every so often.

Regardless of what you have open, you'll see a warning pop up at the bottom.

To defeat the enemy, quickly click on it before five seconds pass.

If you fail to defeat the enemy, your ship will be damaged and you'll lose 2 minutes of time.
